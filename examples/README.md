# Example scripts

Each file matches a lesson in **LEARNING.md**.

**Easiest:** Open a file and click **▶ Run** in Cursor.  
**Terminal (Mac):** `python3 examples/01_variables.py` — use `python3` if `python` says "command not found".

| File | Lesson |
|------|--------|
| `01_variables.py` | 2 – Variables |
| `02_input.py` | 3 – Input |
| `03_math.py` | 4 – Math |
| `04_if_else.py` | 6 – If/else |
| `05_loops_and_lists.py` | 7 & 8 – Loops & lists |
| `06_functions.py` | 10 – Functions |

**Tip:** After running, open the file, change something small (e.g. a name or number), save, and run again to see what happens.
